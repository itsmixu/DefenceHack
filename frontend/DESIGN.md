# Design System: Tactile Noir

A high-contrast, technical aesthetic inspired by film noir, urban cartography, and analog terminal systems. The design focuses on physical-feeling interfaces, precision, and a "tactile" digital experience.

## Visual Identity

### Color Palette
- **Surface**: `#131313` (Deep, matte black)
- **Primary**: `#FFFFFF` (Crisp, high-contrast white)
- **Accent**: `#393939` (Mid-range grays for depth)
- **Overlay**: `rgba(0, 0, 0, 0.5)` with `backdrop-blur` (Glassmorphism for floating elements)

### Typography
- **Headlines**: Technical Sans-Serif (e.g., Inter, bold/black weights)
- **Data/Logs**: Monospaced (e.g., JetBrains Mono, Fira Code)
- **Style**: All-caps labels, generous letter spacing (tracking-wider), and small font sizes for a "dense data" look.

### UI Components & Physicality
- **Floating Panels**: 1/3 screen width, high-contrast borders (1px solid white/10%), backdrop blur.
- **Buttons/Inputs**: Hard-edged (ROUND_FOUR), subtle shadows to create a "pressed" or "etched" look.
- **Indicators**: High-contrast states (Active: #FFFFFF, Inactive: #393939).
- **Backgrounds**: High-contrast, minimalist topographic maps with subtle grain textures.

## Design Principles
1. **High Contrast**: Every element must be either light or dark; avoid muddy mid-tones except for depth.
2. **Technical Density**: Information should feel packed and professional, like a high-end command center.
3. **Tactile Feedback**: Use sharp borders and subtle elevation to give the digital interface a physical presence.
4. **Cinematic Mood**: Leverage whitespace and dark surfaces to create a professional, "noir" atmosphere.
